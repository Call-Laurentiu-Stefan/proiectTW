CleanAll - aplicatie web pentru servicii de spalatorie publica
  1. 1. Abstract

    CleanAll este o aplicatie bazata pe tehnologiile web si este creat pentru a usura oamenilor nevoia de a-si spala rufele.

  2. 2. Introducere

    CleanAll este proiectul nostru la Web Tehnologies. Ofera functionalitatea de a crea o comanda (pentru client), respectiv crearea unui status privind spalarea rufelor(disponibil numai pentru operatori).
